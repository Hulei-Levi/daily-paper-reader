## 摘要
动作质量评估（AQA）旨在从动作视频中客观评估表现质量。现有方法大多遵循“一对一”范式，即为每种动作类型训练一个单独的模型。这种设置限制了实际部署，因为它需要先验的动作类型知识来选择相应模型，并且在不同动作之间泛化能力较差。为了解决这些限制，我们研究了全能型AQA这一具有挑战性的任务，其目标是在单个统一模型中评估异构动作。我们提出了一种新颖的混合动作知识专家（MoAKE）框架，旨在减轻由动作间巨大语义差异导致的负知识迁移。MoAKE学习互补的专家，在共享语义空间中捕获多样化的动作模式，并动态聚合它们的知识以适应输入动作的评估。每个专家都配备了片段感知原型以处理不同的时间长度，同时结合了自适应段内和段间关系建模（AIISRM）模块来建模多粒度时间动态。此外，我们为全能型以及零样本/少样本AQA建立了全面的基准。在三个长期数据集上的大量实验表明，MoAKE在全能设置下显著优于现有方法，同时在零样本/少样本评估下在三个短期数据集上也实现了一致的泛化。代码可在 https://github.com/XuHuangbiao/MoAKE 获取。

## Abstract
Action Quality Assessment (AQA) aims to objectively evaluate performance quality from action videos. Most existing methods follow a ``one-by-one'' paradigm, training a separate model for each action type. This setting limits real-world deployment, as it requires prior action-type knowledge to select the corresponding model and suffers from poor generalization across diverse actions. To address these limitations, we study the challenging task of all-in-one AQA, which aims to assess heterogeneous actions within a single unified model. We propose a novel Mixture of Action Knowledge Experts (MoAKE) framework, designed to mitigate negative knowledge transfer caused by large semantic discrepancies among actions. MoAKE learns complementary experts that capture diverse action patterns within a shared semantic space and dynamically aggregates their knowledge to adapt the assessment to the input action. Each expert is tailored with segment-aware prototypes to handle varying temporal lengths, together with an Adaptive Intra- and Inter-Segment Relationship Modeling (AIISRM) module to model multi-granularity temporal dynamics. Furthermore, we establish comprehensive benchmarks for all-in-one as well as zero/few-shot AQA. Extensive experiments on three long-term datasets demonstrate that MoAKE significantly outperforms existing methods in the all-in-one setting, while also achieving consistent generalization on three short-term datasets under zero/few-shot evaluation. Code is available at https://github.com/XuHuangbiao/MoAKE.

## 1. 论文的核心问题与整体含义（研究动机和背景）
- **核心问题**：现有动作质量评估（AQA）大多采用“一对一”范式，即对每种动作类型单独训练一个模型。这导致三个实际缺陷：(1) 需要先验的动作类型信息才能选择对应模型，无法用于开放域平台；(2) 难以处理混合或未知类型的动作；(3) 在资源稀缺场景下训练和维护多个模型代价高昂。  
- **背景**：尽管已有研究探索跨动作迁移，但直接训练一个统一模型会导致“负知识迁移”——不同动作间巨大的语义差异互相干扰，性能严重下降（平均SRCC下降10.17%）。  
- **研究目标**：提出一种“全能型”（All-in-One）AQA框架，用单一模型评估多种异构动作，并实现正迁移。

## 2. 论文提出的方法论：核心思想、关键技术细节
### 核心思想：混合动作知识专家（MoAKE）
- 采用专家混合（Mixture-of-Experts, MoE）架构，但与传统MoE不同：  
  - 在宏观动作级别上以轻量方式运行（K=3个专家，而非token级稀疏路由）。  
  - 采用**软路由**机制（soft routing），不强制专家严格隔离，而是动态混合互补知识，将语义差异转化为正迁移。
- 整体流程：  
  1. 预训练视觉编码器提取视频特征 X ∈ R^(T×D)。  
  2. 通过 K 个动作知识专家并行处理，每个专家输出 F_k = E_k(X)。  
  3. 轻量软路由器 R(·) 根据视觉特征动态计算权重 w_k（总和为1），得到混合特征 F_mix = Σ w_k F_k。  
  4. 混合特征输入等级感知解码器（grade-aware decoder），生成M个等级表示，再通过MLP预测最终分数 ŷ。

### 关键技术细节
- **动作知识专家（Action Knowledge Experts）**：  
  - 每个专家包含：  
    - 共享适配器（Shared Adapter）：将所有动作映射到共享语义空间。  
    - 片段感知时间聚合（SATA）：使用可学习的段感知原型（N=64个）通过交叉注意力将变长输入聚合为固定长度表示。  
    - **自适应段内和段间关系建模（AIISRM）**：三个并行分支分别建模：  
      * 段内全局关系（沿特征维度的自注意力）  
      * 段间局部关系（多尺度1D卷积 + 动态融合）  
      * 段间全局关系（沿段维度的自注意力）  
    - 输出通过残差连接结合。
- **软路由机制**：  
  - 训练时使用标签平滑的交叉熵损失 L_route 引导路由器，但推理时仅基于视觉特征，无需动作标签。  
- **损失函数**：组合损失 L_total = λ1 L_score + λ2 L_route + λ3 L_div  
  - L_score：主分数回归损失（含辅助专家预测）  
  - L_route：软路由损失（标签平滑）  
  - L_div：等级表示多样性损失（三重损失）
- **数据增广**：以0.6概率将10%的片段替换为其他动作类型的片段，并用90%原分数作为伪标签。

## 6. 论文的主要结论与发现
- MoAKE在全在同一设置下显著超越所有现有方法（平均SRCC 0.753，平均R-ℓ2 2.434），甚至优于One-by-One的SOTA（平均SRCC提升1.2%，R-ℓ2降低28.9%）。

- 软路由至关重要：移除后SRCC下降30%、R-ℓ2上升150%。  
- 专家混合能有效利用互补知识，形成更清晰、可分的特征空间。

## 8. 不足与局限
- **专家数量固定**：当动作类型高度模糊或远超预设专家数时，路由可能无法良好适应。  
- **基准覆盖有限**：主要专注于体育和医疗动作，未涵盖更多样化的日常或工业动作场景。

- **未探索模态缺失场景**：与MCMoE不同，本研究假设视觉模态完整，未考虑模态缺失情况。  
- **泛化风险**：所有实验基于公开数据集，可能存在分布偏差，真实开放场景下的鲁棒性尚未验证。
