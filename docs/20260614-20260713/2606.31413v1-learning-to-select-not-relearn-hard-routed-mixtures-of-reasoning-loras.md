## 摘要
将独立训练的LoRA适配器组合成单个大型语言模型对于多领域适应非常有用，特别是在无法共享原始训练数据的情况下。一种常见的方法是使用MoE风格的专家路由，但对于冻结的预训练适配器，软加权组合可能会改变每个LoRA模块最初训练时使用的单位尺度加法更新。我们提出了**硬路由MoR-LoRA**，这是一个两阶段框架，通过单位尺度的硬选择来组合冻结的推理LoRA专家。首先，使用来自可验证反馈的强化学习独立训练领域特定的LoRA适配器以获得推理专家。然后，冻结所有专家，从中蒸馏推理轨迹，仅训练一个轻量级共享路由器以及一个小型注意力LoRA用于集成。路由器使用硬top-1路由为每个token精确选择一个专家，同时通过直通估计器实现基于梯度的训练。在五个基准、多个模型规模和额外模型家族上的实验表明，硬路由MoR-LoRA保持了专家行为，同时需要的可训练参数远少于软路由混合基线。我们的分析进一步表明，归一化的软混合通常将大部分路由质量集中在一个专家上，这表明硬单位尺度路由为冻结的LoRA专家组合提供了一种简单有效的抽象。

## Abstract
Composing independently trained LoRA adapters into a single large language model is useful for multi-domain adaptation, especially when the original training data cannot be shared. A common approach is to use MoE-style routing over LoRA experts, but for frozen pretrained adapters, soft weighted combinations can change the unit-scale additive update under which each LoRA module was originally trained. We propose \textbf{Hard-Routed MoR-LoRA}, a two-stage framework for composing frozen reasoning LoRA experts through unit-scale hard selection. First, domain-specific LoRA adapters are trained independently using reinforcement learning from verifiable feedback to obtain reasoning experts. Then, all experts are frozen, reasoning traces are distilled from them, and only a lightweight shared router together with a small attention LoRA is trained for integration. The router selects exactly one expert per token using hard top-1 routing, while a straight-through estimator enables gradient-based training. Experiments across five benchmarks, multiple model scales, and additional model families show that Hard-Routed MoR-LoRA preserves expert behavior while requiring substantially fewer trainable parameters than soft-routing mixture baselines. Our analysis further shows that normalized soft mixtures often concentrate most routing mass on a single expert, suggesting that hard unit-scale routing provides a simple and efficient abstraction for frozen LoRA expert composition.

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：如何将多个独立训练的、冻结的LoRA适配器（专家）组合成一个统一的大语言模型（LLM），用于多领域适应，同时保持每个专家的原始行为（单位尺度加法更新）。
- **背景**：LoRA是一种参数高效微调（PEFT）方法，通过在冻结骨干网络上添加低秩可训练矩阵来适应下游任务。在实际中，不同团队可能独立训练LoRA适配器，但由于隐私或监管限制无法共享原始数据，只能发布训练好的LoRA模块。因此，需要一种方法将这些独立训练的专家动态组合起来，以支持异构任务。
- **现有方法的不足**：常用的MoE软路由（soft routing）在组合冻结的LoRA专家时存在**尺度不匹配**问题：每个LoRA专家训练时假定其更新是单位加法（即权重为1），而软路由使用加权求和（权重小于1），改变了有效更新幅度，破坏了专家的原始行为。先前工作（如LoRA-Mixer）通过额外微调专家来补偿这一失配，但这增加了可训练参数和计算成本，并可能改变专家知识。
- **研究动机**：提出一种更直接的方法，通过**硬路由（hard routing）** 在逐token级别上精确选择一个专家，并以单位尺度应用，从而保持专家冻结状态的同时进行有效组合。

## 2. 方法论：核心思想、关键技术细节

### 核心思想
两阶段框架：**推理获取与专家选择解耦**。
- **阶段一**：对每个领域独立训练LoRA专家，采用**可验证反馈强化学习（RLVF）** 训练推理能力。
- **阶段二**：冻结所有专家，通过蒸馏从专家收集推理轨迹，仅训练一个轻量级共享路由器和一个小型注意力LoRA，学习在每个token上选择哪个专家。

### 关键技术细节
1. **尺度失配问题**：标准MoE软路由的输出为 \(W_{\text{updated}} = W_{\text{pretrained}} + \sum_{i=1}^k \omega_i A_i B_i\)，其中\(\omega_i\)是路由概率，改变了LoRA更新的幅度。而LoRA训练假设单位贡献（\(\omega_i=1\)）。
2. **硬路由与直通估计器（STE）**：
   - 对每个token \(x\)，计算路由logits \(G(x)\)，得到softmax概率\(P\)，然后选择概率最大的专家索引\(j = \arg\max_i P_i\)。
   - 产生one-hot路由向量\(Y_{\text{hard}}\)，最终输出为\(E_j(x)\)（精确应用选中专家）。
   - 前向传播使用硬决策；反向传播通过STE近似梯度：\(Y = \text{SG}(Y_{\text{hard}} - P) + P\)，使得梯度通过软概率\(P\)传播。
   - 优点：保持单位尺度，专家冻结，仅训练路由器（共享线性层）和注意力LoRA（小秩）。
3. **两阶段训练流程**：
   - **阶段一：训练推理LoRA专家**：每个数据集独立训练一个LoRA适配器，使用RLVF（采用GRPO算法）优化可验证的正确性。目标函数为off-policy token-level GRPO：\(\mathcal{L}_{\text{off}}(\theta) = -\mathbb{E}_x\left[\frac{1}{N}\sum_{i=1}^N A_i \sum_{t=1}^{T_i} w_{i,t} \log \pi_\theta(y_{i,t}|x, y_{i,<t})\right]\)，其中\(A_i = r_i - \bar{r}\)，\(w_{i,t}\)为重要性权重。
   - **阶段二：通过蒸馏进行混合集成**：
     - 使用训练好的专家生成推理轨迹（每个领域最多1000个样本）。
     - 冻结所有专家，训练共享线性路由器和注意力LoRA（参数极小），使用标准监督学习（SFT）最大化蒸馏数据上的似然。

## 6. 主要结论与发现

1. **RLVF产生更强的推理专家**：对于中等和大规模模型（3B、8B），RLVF显著优于SFT，特别是在推理密集型任务上；SFT可能削弱大模型已有的CoT行为。但1B模型因容量不足，RLVF收益有限。
2. **硬路由保留专家行为，参数效率高**：在冻结专家的情况下，硬路由（Hard-Routed MoR-LoRA）达到与软路由基线相当或更好的平均性能，但训练参数大幅减少（3B模型：73M vs LoRAMixer的606M；8B模型：109M vs 1.133B）。
3. **软路由实际行为接近硬路由**：分析归一化top-2软路由的权重分布，发现平均主路由权重约为0.71，熵较低，说明软混合经常简化为近似的单专家选择。
4. **集成阶段主要学习路由而非推理**：SFT训练蒸馏轨迹已足够，使用RLVF收益很小；训练数据仅需1000样本就能接近全数据性能。
5. **混合领域场景下，逐token路由更具灵活性**：在包含多领域问题的输入中，prompt-level路由因只能选一个专家而受限，而逐token硬路由可动态切换专家，获得更好平均性能。

## 7. 优点

- **概念简洁**：直接解决尺度失配问题，通过硬选择保持专家原始行为，无需重新训练专家。

- **两阶段解耦**：先独立训练推理专家，再学习路由，使得推理能力获取和专家选择分离，更灵活。
- **全面实验**：覆盖多模型规模、多领域、多基线、消融实验、迁移测试，结果稳健。
- **分析深入**：通过对软路由权重的分布分析，揭示了软路由在实践中趋向硬路由行为，为硬路由的有效性提供了理论支撑。
